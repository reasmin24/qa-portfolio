## Automation Testing

This folder contains test automation work created to validate web application
functionality efficiently and reliably.

### Tools & Technologies:
- Playwright
- Python
- PyTest
- GitHub

### Includes:
- Automated test scripts for critical user flows
- Page Object Model (POM) structure
- Reusable test utilities and helpers
- Assertions and basic reporting
- Sample test execution results

### Test Coverage:
- Smoke and regression test scenarios
- UI validation and user interaction flows
- Basic error handling and waits

### Objective:
Reduce manual testing effort by automating repetitive test scenarios and
ensure faster feedback on application quality.
